# الزبدة - ملخص يوتيوب

تطبيق ويب لتلخيص فيديوهات يوتيوب باستخدام الذكاء الاصطناعي.

## 🚀 النشر على Render.com

### الخطوات:

1. **أنشئ حساب على [Render.com](https://render.com)**

2. **ارفع المشروع على GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/Krtosh/alzobda.git
   git push -u origin main
   ```

3. **من Render Dashboard:**
   - اضغط **"New +"** → **"Web Service"**
   - اختر Repository من GitHub
   - الإعدادات تلقائية من `render.yaml`

4. **أضف Environment Variables:**
   - `OPENAI_API_KEY` = مفتاح OpenAI
   - `YOUTUBE_API_KEY` = مفتاح YouTube Data API

5. **اضغط "Create Web Service"**
